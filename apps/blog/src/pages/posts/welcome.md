# Welcome to Minions Retainers

We're excited to introduce **Minions Retainers** — Recurring service agreements, care plans, monthly retainers, and subscription management.

## What's Included

- **TypeScript SDK**: `npm install @minions-retainers/sdk`
- **Python SDK**: `pip install minions-retainers`
- **CLI**: `npm install -g @minions-retainers/cli`

## Getting Started

Check out the [documentation](https://retainers.minions.help) for a full guide.
