---
title: Installation
description: Install Minions Retainers packages
---

## TypeScript / Node.js

```bash
npm install @minions-retainers/sdk minions-sdk
```

## Python

```bash
pip install minions-retainers
```

## CLI

```bash
npm install -g @minions-retainers/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
