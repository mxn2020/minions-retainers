# Minions Retainers

Recurring service agreements, care plans, monthly retainers, and subscription management

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-retainers/sdk`) — Core library for Node.js
- **Python SDK** (`minions-retainers`) — Python bindings
- **CLI Tool** (`@minions-retainers/cli`) — Command-line interface
- **Web App** — Interactive playground at [retainers.minions.wtf](https://retainers.minions.wtf)
